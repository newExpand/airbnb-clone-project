## 프로젝트 제목

Airbnb 클론 코딩

## 프로젝트 설명

Airbnb 클론 프로젝트는 React, Tailwind, Prisma, MongoDB, NextAuth를 사용하여 제작된 웹 어플리케이션입니다. 세계적으로 많이 찾는 Airbnb의 디자인과 기능이 마음에 들어 제 방식으로 재구성 해보았습니다.

## Installation

### 저장소 복제

```
git clone https://github.com/newExpand/airbnb-clone-project.git
```

### 패키지 설치

```
npm i
```

### 앱 시작

```
npm run dev
```
